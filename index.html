How it's built

Powered by the HTML5 Canvas API

SketchSync draws every stroke directly onto a <canvas> element using vanilla JavaScript — no frameworks, no libraries, no build step. When you press down, we start tracking a path of points; as you move, line segments connect those points in real time using lineTo() and stroke().

Shapes work differently: instead of permanently committing pixels on every frame, SketchSync redraws the entire history on a buffer and previews the shape on top, so your rectangle or arrow follows your cursor cleanly until you release the mouse. That snapshot-based approach is also what makes undo and redo possible — every completed stroke is pushed onto a history stack you can travel through.

  • Canvas resizes responsively while preserving your drawing.
  • Touch, mouse, and stylus input are all normalized into one coordinate system.
  • Your theme preference is remembered using localStorage.
